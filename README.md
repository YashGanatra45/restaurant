🍽️ Restaurant Website
A modern and responsive restaurant website designed to showcase menu items, allow user interaction, and provide a smooth browsing experience.

🚀 Features

🔐 User Login & Signup system

🍔 Attractive food menu display

🛒 Order placement interface

💳 Multiple payment options

📱 Responsive design (works on mobile & desktop)

🎨 Clean and user-friendly UI


🛠️ Technologies Used

HTML

CSS

JavaScript

PHP

MySQL

⚙️ How to Run the Project

👉 Option 1: Run Locally
Download or clone the repository
Extract the ZIP file (if downloaded)
Open index.html in your browser

👉 Option 2: Using Local Server (for backend projects)
Install XAMPP / WAMP
Place project folder inside htdocs
Start Apache & MySQL
Open browser and go to:
http://localhost/restaurant
